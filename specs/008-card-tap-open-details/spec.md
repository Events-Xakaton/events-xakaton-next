# Спека 008: Открытие деталей по тапу на карточку

## Цель

Сделать поведение карточек ивентов и клубов более ожидаемым на мобильных
экранах: открывать детали не только через кнопку `Детали`, но и по тапу на
саму карточку.

## Проблема

1. Сейчас переход в детали в ряде карточек доступен только через отдельную
   кнопку `Детали`/иконку.
2. Пользователь ожидает, что тап по карточке целиком откроет тот же экран
   деталей.
3. На разных экранах поведение карточек отличается.

## Требования

1. Для всех карточек ивентов и клубов, где есть действие открытия деталей,
   тап по свободной области карточки должен открывать те же детали.
2. Кнопки действий внутри карточек (`Участвовать`, `Вступить`, `Детали` и т.п.)
   должны сохранить текущее поведение и не вызывать побочных действий.
3. Изменение должно быть применено консистентно на всех экранах, где
   используются карточки ивентов и клубов:
   - лента (home);
   - профиль (account);
   - секция ивентов внутри деталей клуба (club details).

## Non-goals

- Не менять бизнес-логику join/unjoin/join-club.
- Не менять маршрутизацию и структуру экранов деталей.
- Не менять визуальный дизайн карточек, кроме улучшения интерактивности.
