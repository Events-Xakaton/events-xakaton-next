# Quickstart: Required Field Markers on Create Screens

## Preconditions

- Запущен frontend проект `events-xakaton-next`.
- Открыт экран `Создать`.

## Scenario 1: Event form markers

1. Переключиться на вкладку создания мероприятия.
2. Проверить, что поля `Название`, `Описание`, `Локация`, `Начало`, `Окончание` отмечены `*`.
3. Проверить, что необязательные поля (`Лимит`, `Для детей`, `Возраст`, `Минимальный уровень`, `Клуб`) без `*`.
4. Проверить наличие подсказки `* Обязательные поля`.

## Scenario 2: Club form markers

1. Переключиться на вкладку создания клуба.
2. Проверить, что `Название` и `Описание` отмечены `*`.
3. Проверить наличие подсказки `* Обязательные поля`.

## Scenario 3: Button behavior unchanged

1. На форме мероприятия оставить одно обязательное поле пустым -> кнопка `Создать мероприятие` неактивна.
2. Заполнить все обязательные поля корректно -> кнопка активируется.
3. На форме клуба оставить `Название` или `Описание` пустым -> кнопка `Создать клуб` неактивна.
4. Заполнить `Название` и `Описание` -> кнопка активируется.
