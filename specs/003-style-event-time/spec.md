# Спека 003: Единый стиль времени на карточках ивентов

## Цель

Сделать отображение времени на карточках ивентов в едином стиле кнопки
`Другой фон` с экрана создания мероприятия.

## Источник стиля

- Базовый стиль: `preview-card__change-bg-button`
- Файл-эталон: `src/shared/components/styles/preview-card.css`

## Область применения

Время должно быть оформлено в новом стиле на всех местах, где ивенты
показываются именно карточками:

1. `src/entities/event/ui/event-feed-card.tsx` (главная лента)
2. `src/views/account/ui/profile-event-card.tsx` (профиль)
3. `src/views/club-details/index.tsx` (список карточек ивентов клуба)

## Требования к UI

1. Время отображается в виде pill-chip:
   - `inline-flex`
   - высота `h-7`
   - `rounded-full`
   - `backdrop-blur-sm`
   - компактный текст (11px, semi-bold, tracking)
   - иконка времени слева
2. На темных/градиентных карточках используется прозрачный светлый вариант
   (как `Другой фон`).
3. На светлой карточке в клубе используется адаптированный светлый surface-вариант,
   но с той же геометрией и типографикой.
4. Логика данных и формат времени не меняются.

## Non-goals

- Не меняем формат даты/времени.
- Не меняем логику фильтрации/загрузки ивентов.
- Не меняем другие элементы карточек (кнопки, заголовки, участники).

## Изменяемые файлы

- `src/app/globals.css`
- `src/entities/event/ui/event-feed-card.tsx`
- `src/views/account/ui/profile-event-card.tsx`
- `src/views/club-details/index.tsx`
