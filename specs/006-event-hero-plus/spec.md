# Спека 006: Кнопка участия на hero-карточке ивента

## Цель

Добавить быстрый action участия прямо на градиентную hero-карточку ивента в
экране деталей события, чтобы пользователь мог записаться без прокрутки до
нижней sticky-панели.

## Проблема

1. На экране деталей ивента action участия находится только в нижней панели.
2. Пользователь ожидает увидеть `+`-кнопку участия прямо на карточке ивента,
   как в секции `Ивенты` внутри карточки клуба.

## Требования

1. В `PreviewCard` ивента (hero-блок с градиентом) добавить кнопку участия:
   - иконка `+`, без текста;
   - визуально и по размеру консистентна с `+`-кнопкой в карточках ивентов
     клуба.
2. Кнопка должна использовать существующий join-flow экрана `EventDetails`.
3. Состояния кнопки:
   - если пользователь уже участвует: отображается состояние `joined`
     (иконка-check/позитивное состояние);
   - если участие невозможно (архив/ограничения), кнопка не должна давать
     некорректное действие.
4. Текущее поведение sticky-кнопок и остальных действий на экране не должно
   ломаться.

## Файлы

- `src/views/event-details/index.tsx`
- при необходимости: `src/shared/components/preview-card.tsx`

## Non-goals

- Не менять API и контракты join/unjoin.
- Не перерабатывать sticky actions внизу экрана.
